Understanding Shortest Path Algorithms

Finding the shortest path between nodes in a weighted graph is fundamental to network optimization.

Dynamic Programming Approach

Works on DAGs (Directed Acyclic Graphs) by processing nodes in topological order.

dist[v] = min(dist[u] + weight(u,v)) for all edges (u,v)
Bellman-Ford Algorithm
  • Works with negative edge weights
  • Detects negative cycles
  • Relaxes all edges V-1 times
For each edge (u,v):
if dist[u] + weight(u,v) < dist[v]:
  dist[v] = dist[u] + weight(u,v)
Negative Cycle Detection

After V-1 iterations, if any edge can still be relaxed, a negative cycle exists.

Choosing the Right Algorithm

DP: For acyclic graphs, faster O(V+E)

Bellman-Ford: For graphs with negative weights, O(VE)
