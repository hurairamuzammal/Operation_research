Assignment Problem (Hungarian Method)

Understanding the Assignment Problem

The Assignment Problem is a special case of the Transportation Problem where each worker must be assigned to exactly one task, minimizing total cost.

Hungarian Method Steps
  • Step 1: Subtract row minimum from each row
  • Step 2: Subtract column minimum from each column
  • Step 3: Cover all zeros with minimum number of lines
  • Step 4: If lines < n, modify matrix and repeat
  • Step 5: Make optimal assignment using zeros
Matrix Modification (Step 4)
1. Find minimum uncovered element (k)
2. Subtract k from all uncovered elements
3. Add k to elements at line intersections
4. Covered elements remain unchanged
Optimality Condition

Optimal when number of lines covering all zeros = number of rows (n)

Maximization Problems

Convert to minimization: Subtract all values from the maximum value in the matrix, then solve normally.
