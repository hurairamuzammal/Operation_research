Understanding the Simplex Method

The Simplex algorithm solves linear programming problems by moving along the edges of the feasible region to find the optimal vertex.

Standard Form Requirements
  • All constraints must be equations (use slack/surplus variables)
  • All variables must be non-negative
  • RHS values should be non-negative
Simplex Tableau Operations
1. Choose entering variable: Most negative coefficient in Z-row (max)
2. Choose leaving variable: Minimum ratio test (RHS / pivot column)
3. Pivot: Make pivot element = 1, all other column entries = 0
When to Use Big-M Method
  • When constraints have ≥ or = signs
  • Artificial variables are added with coefficient M (very large)
  • Goal: Drive artificial variables out of the basis
  • If any artificial variable remains positive → Infeasible
Optimality Conditions
  • Maximization: Optimal when all Z-row coefficients ≥ 0
  • Minimization: Optimal when all Z-row coefficients ≤ 0
  • Unbounded: No positive ratio in minimum ratio test
Branch and Bound (Integer Solutions)

When integer solutions are required, we solve LP relaxation first, then branch on fractional variables, bounding with LP solutions to prune the search tree.
