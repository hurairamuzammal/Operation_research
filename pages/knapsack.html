Understanding the Knapsack Problem

The Knapsack Problem is a classic optimization problem where you must select items with given weights and values to maximize total value without exceeding a weight capacity.

Types of Knapsack Problems
  • 0/1 Knapsack: Each item can only be taken once (binary choice)
  • Bounded Knapsack: Each item has a limited quantity available
  • Unbounded Knapsack: Unlimited copies of each item available
Dynamic Programming Approach

We build a table dp[i][w] where each cell represents the maximum value achievable using items 1 to i with capacity w.

dp[i][w] = max(dp[i-1][w], dp[i-1][w-wᵢ] + vᵢ)

Where: wᵢ = weight of item i, vᵢ = value of item i
How to Read the DP Table
  • Each row represents considering items 1 through i
  • Each column represents a capacity from 0 to W
  • The final answer is in cell dp[n][W]
  • Backtrack from this cell to find which items were selected
Example

Items: (weight=2, value=3), (weight=3, value=4), Capacity=5

Solution: Take both items → Total weight=5, Total value=7
